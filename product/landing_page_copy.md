## FAQ

**Is this just another AI resume tool?**
No. The core score is deterministic skill-overlap math — reproducible and auditable, not a black-box model. The same inputs always give the same result. AI is an *optional* layer that only polishes the suggestion wording.

**Does it work offline / without paying for an API?**
Yes. By default it runs a built-in deterministic provider — no API key, no cost, no data sent anywhere. You can add an OpenRouter key later if you want LLM-tailored suggestions.

**Will it rewrite my resume for me?**
It tells you precisely what's missing and gives actionable suggestions. You stay in control of the wording — which keeps it honest and keeps it *yours*.

**How is the fit score calculated?**
`matched required skills ÷ total required skills in the job description`, scaled to 0–100. Simple, transparent, and explainable on every result.

**Is my resume data private?**
In default (deterministic) mode nothing leaves your machine. If you enable the optional LLM suggestions, only the text needed for tips is sent to the configured provider. You choose.

**Is it really free?**
The open-source core is free to run yourself. Paid tiers add convenience, hosted access, and team features — see pricing.
